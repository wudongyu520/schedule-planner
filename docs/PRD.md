# 时间块日程规划系统 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
时间块日程规划系统 (Time Block Schedule Planner)

### 1.2 产品理念
基于"时间块管理法"的现代日程规划工具。将一天24小时作为时间轴，用户可以自由创建任意时间段的功能区（时间块），将任务拖拽到功能区中安排执行。通过可视化的时间块管理，帮助用户专注当下，提高时间利用效率。

### 1.3 核心价值
- **自由时间块**：任意时间段，灵活组合，不受固定时辰限制
- **拖拽式规划**：可视化拖拽创建、调整时间块，安排任务
- **当下专注**：当前时间高亮提醒，实时显示剩余时间
- **及时止损**：时间块结束自动作废未完成任务，不拖累后续计划
- **复盘改进**：日志记录失败原因，持续优化时间管理

## 2. 核心概念

### 2.1 时间轴 (Timeline)
- 一天 = 24小时 = 1440分钟
- 垂直排列，从上到下表示时间流逝
- 最小粒度：5分钟（可配置）
- 左侧显示时间刻度（每小时一个刻度）

### 2.2 功能区 / 时间块 (TimeBlock)
- 一天中的任意时间段（开始时间 ~ 结束时间）
- 可设置名称、背景颜色
- 例如：
  - "晨间学习"：07:00 - 09:00（2小时）
  - "深度工作"：09:30 - 11:30（2小时）
  - "午休"：12:00 - 13:30（1.5小时）
- 功能区是任务的容器
- 可以自由拖拽调整位置和大小

### 2.3 任务 (Task)
- 可拖拽的待办事项
- 属性：标题、预估时长、优先级、标签
- 可以放置在功能区内
- 一个功能区可以放多个任务

### 2.4 当前时间线 (Now Line)
- 实时显示当前时间的水平线
- 精度：分钟级（可切换秒级）
- 用于判断当前处于哪个功能区
- 显示当前功能区剩余时间

## 3. 功能需求

### 3.1 视图模块

#### 3.1.1 周视图（主视图）
- 布局：左侧任务面板 + 7列（周一至周日）
- 每列：一天的时间轴（0:00 - 24:00）
- 时间块在时间轴上按位置和高度显示
- 支持上下滚动查看完整一天
- 支持左右滑动切换周
- 顶部：日期导航栏（上一周/下一周/今天）

#### 3.1.2 日视图
- 单列展示当天时间轴
- 任务面板在侧边抽屉
- 更详细的任务展示
- 时间刻度更精细

#### 3.1.3 月视图
- 日历格子展示
- 每格显示当天任务数量概览
- 点击某天进入日视图
- 月份切换

### 3.2 时间块（功能区）管理

#### 3.2.1 创建时间块
- 方式一：在时间轴空白处拖拽选择时间段
- 方式二：点击空白处弹出创建表单，输入开始/结束时间
- 设置名称
- 设置背景颜色（预设颜色 + 自定义）
- 可选：保存为模板

#### 3.2.2 编辑时间块
- 修改名称
- 修改颜色
- 调整时间段：
  - 拖拽顶部：调整开始时间
  - 拖拽底部：调整结束时间
  - 拖拽中间：整体移动
- 最小粒度：5分钟

#### 3.2.3 删除时间块
- 删除后内部任务移回任务面板
- 确认提示

#### 3.2.4 时间块重叠处理
- 不允许时间块重叠
- 拖拽时检测碰撞，阻止重叠
- 或自动调整相邻时间块（可选）

#### 3.2.5 时间块模板
- 预设常用模板（工作、学习、休息等）
- 支持保存自定义模板
- 一键应用模板到某天/某周
- 模板管理（增删改）

### 3.3 任务管理

#### 3.3.1 创建任务
- 输入任务标题
- 设置预估时长（分钟/小时）
- 设置优先级（高/中/低）
- 添加标签（可选）
- 添加备注/描述（可选）

#### 3.3.2 任务面板
- 显示所有未安排的任务
- 按优先级排序
- 支持搜索过滤
- 支持按标签筛选
- 任务分组：高优先级、普通任务、已完成等

#### 3.3.3 任务拖拽安排
- 从任务面板拖拽到时间块
- 时间块内调整任务顺序
- 跨时间块拖拽移动任务
- 拖拽回任务面板取消安排
- 拖拽时显示任务时长预览

#### 3.3.4 任务完成
- 点击勾选完成任务
- 完成的任务显示删除线 + 移至底部
- 可撤销完成

#### 3.3.5 任务编辑/删除
- 点击任务打开详情弹窗
- 修改任务属性
- 删除任务（确认提示）

### 3.4 时间线与时间块切换

#### 3.4.1 当前时间线
- 醒目的水平线标记当前时间
- 精度：分钟（可切换秒级）
- 随时间自动移动
- 颜色：红色/橙色，高对比度
- 显示时间标签（当前具体时间）

#### 3.4.2 当前功能区高亮
- 当前所处的时间块高亮显示
- 显示剩余时间（如"还剩 45 分钟"）
- 时间块边框加粗/变色

#### 3.4.3 时间块切换机制
- 当时间线进入下一个时间块时：
  - 上一个时间块的所有未完成任务自动"过期作废"
  - 作废的任务移回任务面板
  - 高优先级任务标记为"待重排"（红色标记）
  - 常规任务直接移回待办列表
- 触发日志记录

#### 3.4.4 任务作废规则
- **高优先级任务**：移回任务面板顶部，标记红色"待重排"
- **中/低优先级任务**：移回任务面板正常位置
- 所有作废任务记录到当日日志，可备注原因

### 3.5 日志系统

#### 3.5.1 当日日志
- 记录所有作废的任务
- 记录完成的任务
- 记录时间块变更
- 按时间顺序排列

#### 3.5.2 失败原因备注
- 任务作废时可添加原因备注
- 预设原因：时间不够、突发事项、拖延、优先级调整、被打断等
- 支持自定义原因

#### 3.5.3 日志查看
- 按天查看日志
- 统计完成率、作废率
- 趋势图表（周/月）
- 时间块使用统计

### 3.6 用户系统

#### 3.6.1 注册/登录
- 邮箱注册
- 密码登录
- 记住登录状态
- 找回密码

#### 3.6.2 数据同步
- 云端存储
- 多设备同步
- 离线可用（本地缓存）
- 冲突处理（最后写入胜出）

## 4. 非功能需求

### 4.1 性能要求
- 页面首屏加载 < 2s
- 拖拽操作流畅，60fps
- 数据保存延迟 < 500ms
- 时间线更新无明显卡顿

### 4.2 兼容性
- 支持 Chrome / Safari / Firefox / Edge 最新版本
- 响应式设计，支持平板
- 最低支持宽度：1024px

### 4.3 数据安全
- 用户数据加密存储
- 密码加盐哈希（bcrypt）
- HTTPS 传输
- SQL 注入防护

## 5. 交互设计要点

### 5.1 时间块拖拽体验
- 创建：在空白时间轴上按下拖拽，松手即创建
- 调整大小：拖拽上下边缘，显示时间预览
- 移动：拖拽中间部分，整体移动
- 碰撞检测：接近其他时间块时吸附/阻止

### 5.2 任务拖拽体验
- 拖拽时有半透明预览
- 放置区域高亮提示
- 放置时显示插入位置指示线
- 松手有平滑动画

### 5.3 时间线
- 红色实线，醒目但不刺眼
- 时间标签显示当前精确时间
- 悬浮时可显示详细时间
- 平滑移动动画

### 5.4 颜色系统
- 时间块颜色：柔和的低饱和度色系（蓝、绿、紫、橙、粉等）
- 优先级颜色：
  - 高：红色系
  - 中：黄色系
  - 低：绿色系
- 时间线：鲜红色
- 当前时间块：边框高亮
